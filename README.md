## Run Locally

Clone the project

```bash
  git clone https://github.com/agustinlopez23/SelectArgGeoRefApi
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
